# Kambing

Kartu Bimbingan

## How To Use

1. Install all requirements
```bash
pip install -r requirements
```
2. Change filename from **dawet.cp37-win_amd64.pyd** to **dawet.pyd**
3. Change filename from **kambing.cp37-win_amd64.pyd** to **kambing.pyd**
4. Run the program
```bash
python main.py
```
